# Helpers 🔧

## PowerShell 🧙‍♂️

### Profile 🪄
Open your powershell profile file with
```ps 
notepad $PROFILE
```

In this file, you can paste the functions you want in it.

After that, you can do the following command to update your powershell with new profile configuration 
```ps
. $PROFILE
```
